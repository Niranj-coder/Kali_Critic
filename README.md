# Kali_Critic

A tool which helps find what's wrong with the output if a Kali tool is used.

### Tools supported (as of now)

1. nmap  
2. sqlmap  
3. metasploit  

### How to use

Just run:  
`python kcritic.py`

Paste your tool output + Ctrl+D (or Ctrl+Z + Enter on Windows).

### NOTE

Only three tools are supported as of now. I'll update more tools as we go.
